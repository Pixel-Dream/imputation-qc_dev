Dev Repo for imputation QC tool

## Installation

You can install the development version of imputationqc from GitHub:

``` r
# install.packages("devtools")
devtools::install_github("Pixel-Dream/imputation-qc_dev")
```

## Overview

The imputationqc package provides tools for quality control of genetic imputation results. This package helps researchers validate and assess the quality of imputed genotype data.

## Development

This package is set up for R package development with the following structure:

- `R/` - R source code
- `man/` - Documentation files
- `tests/testthat/` - Unit tests
- `vignettes/` - Long-form documentation
- `data-raw/` - Scripts for preparing data
